# VRPPC
It's a shared repository for VRPPC. 


# To run the program compile below mentioned files:

1. function.cpp
2. main1.cpp
3. initsolcopy.cpp
4. newTwoOpt.cpp
5. newalgocopy.cpp

# For example:

g++ main1.cpp function.cpp initsolcopy.cpp newTwoOpt.cpp newalgocopy.cpp -o main

# To run the program:

./main [text file name]

# For example:

./main CE-01.txt

or 

./main G-01.txt
